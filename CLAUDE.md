# Project rules

## Writing
- **No em dashes anywhere.** Never use `—` (or `--`) in copy, UI text, comments, commit
  messages, or anything else. Rewrite with periods, commas, "and"/"but", parentheses, or a
  middot `·` for separators. Keep prose plain and human.
- **"ImagineArt"** is always written ImagineArt. Never uppercase it to IMAGINEART. Avoid
  CSS `text-transform: uppercase` on text containing the brand name.

## Product facts
- **Film Studio is a model-agnostic shell.** It does not "use 28 models" or expose a bench of
  models. It routes every shot to the best possible cinematic image and video model available.
  (The model-count figures describe the broader ImagineArt suite, not Film Studio.)

## Design
- Match Figma fonts, sizes, and spacing verbatim. No substitutes.
- **Never use JetBrains Mono.** Do not inject new fonts. Use only the fonts already in the
  system (Satoshi, Inter, Geist, Geist Mono, DM Mono). Where a mono/eyebrow style is needed,
  use **Inter** (or DM Mono where the design already uses it), never JetBrains.
